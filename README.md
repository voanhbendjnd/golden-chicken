🐔 Golden Chicken – Online Food Ordering Website
📌 Introduction

Golden Chicken là hệ thống website đặt món ăn online dành cho cửa hàng kinh doanh Gà rán, Combo, Thức ăn nhanh tại Cần Thơ.

Hệ thống giúp khách hàng đặt món nhanh chóng, theo dõi đơn hàng và giúp cửa hàng quản lý bán hàng hiệu quả.

🎯 Objectives

Xây dựng website đặt món trực tuyến

Quản lý đơn hàng theo thời gian thực

Hỗ trợ thanh toán COD & VNPAY

Thống kê và báo cáo doanh thu

Giảm sai sót khi ghi đơn thủ công
